# **App Name**: Blauberia

## Core Features:

- User Authentication Check: Upon arrival, the system checks if a user is currently logged in. If not, the user is redirected to the sign-up page.
- Sign-Up Page UI: A clear and intuitive user interface for new users to register an account.
- Personal Information Capture: Form fields to securely capture date of birth and gender during registration.
- Unique Username Input: Allows users to input a username that must be unique and contain no emojis, with visual feedback for validation.
- Secure Password Creation: Enables users to set a password with a minimum length of 8 characters, ensuring account security.
- Terms of Service Agreement: A clear indication and method for users to agree to the terms of service during the sign-up process.
- Login Page Redirection: A prominent link on the sign-up page ('Already have an account? Login!') to redirect users to the (future) login page.

## Style Guidelines:

- Primary color: A vibrant digital blue (#2E93F7) for interactive elements and brand emphasis, reflecting innovation and imagination.
- Secondary color: A deep, rich black (#000000) for primary text, bold accents, and key interactive components, conveying sophistication and strong presence.
- Background color: A clean, soft white (#FFFFFF) to provide an expansive canvas and ensure readability, maintaining a modern and spacious feel.
- Tertiary color: A neutral medium grey (#6B7280) for secondary text, borders, and subtle UI elements, providing balance and visual hierarchy without overpowering the main colors.
- Headline font: 'Space Grotesk', a sans-serif, for a tech-forward and imaginative feel. Body text font: 'Inter', a grotesque-style sans-serif, for readability and a modern, objective presentation of information.
- Minimalist and clean vector icons with a sharp, geometric design to align with the platform's digital and futuristic identity, emphasizing clarity in user actions.
- A responsive, centered form layout on registration pages, ensuring focus on user input. Ample white space to reduce cognitive load and enhance the clean aesthetic.
- Subtle, fluid transitions for form field focus and button hover states. Small, encouraging feedback animations upon successful data entry to enhance user experience.
